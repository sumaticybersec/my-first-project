 Coffee Shop Billing System

A simple GUI-based billing system built using Python and Tkinter.

---

 Features

- Generate bill with total amount
- Display date & time
- Save bill to file ("bill.txt")
- Reset inputs

---

 Tech Stack

- Python
- Tkinter

---

 Run Project

python coffee_bill.py
